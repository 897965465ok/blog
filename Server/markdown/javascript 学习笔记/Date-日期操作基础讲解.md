# Date-日期操作基础讲解

> date 是日期类， 通过他可以对时间进行出处理


```javascript
 var time = new Date();//=>获取当前客户端本机时间
(当前获取的时间不能作为重要参考依据)
//获取的结果是一个日期个数的对象：2018-04-09T13:37:26.660Z
typeof new Date()->"Object"
//=>time.getFullYear()  获取四位整数年
//=>time.getMonth()  获取月（0~11代表1~12月）
//=>time.getDate()  获取日
//=>time.getDay() 获取星期(0~6 代表周日~周六)
//=>time.getHours()  获取小时
//=>time.getMinutes()  获取分钟
//=>time.getMilliseconds( ) 获取毫秒 
//=>time.getTime() 获取当前日期 获取当前日期距离"1970-01-01-00:00:00"毫秒差
```

```
var time = new Date("4/12/2018 17:36:00"); //=>当new Date中传递一个时间格式的字符串,相当于把这个字符串转成为标准的时间对象格式(转换吼可以调用上面我们讲的那些方法)
//=>时间格式字符串
"2017-10-22"(ie不识别)
"10/22/2017 16:15:34"(ie可以识别)
1508645465766354654(如果传递的是距离1970年的那个毫秒差，也是可以识别转换的,只能是数字不能是字符串)
new Date(150864546576610)//=>Date 6750-09-16T10:29:36.610Z
```

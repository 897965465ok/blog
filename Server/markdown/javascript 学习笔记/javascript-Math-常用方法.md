# javascript-Math-常用方法

```
Math.ads(-12)//=>获取绝对值 12 
向上取整  ceil中文意思:天花板
{
Math.ceil(-1.9)// -1
Math.ceil(1.9)// 2
Math.ceil(-0.9)// -0
Math.ceil(0.9)// 1
} 
向下取整 floor 中文意思:地板
{
Math.floor(11.3)// 11
Math.floor(19.3)// 19
Math.floor(-12.9)//-13
Math.floor(-11.3)//-12
}
{
Math.round()  四舍五入 中文意思：圆
ps:正数5及5以上往上
ps:负数5及5以下往下
}
Math.random(1,10)//  获取0到1的随机数 中文意思:随机数0.8586596773319286

Math.max()//获取一组元素中的最大值
Math.min()//获取一组元素中的最小值

Math.PI  // 获取圆周率 
Math.pow() // 获取一个值的多少次暮
Math.sqrt() // 获取一个值的根开平方
```

## 封装性

```golang
//给父类添加吃的方法
func (node *Porson) eat(food string) {
	fmt.Printf("%s在吃%s\n", node.name, food)
}
//给父类添加喝的方法
func (node *Porson) drink(water string) {
	fmt.Printf("%s在喝%s\n", node.name, water)
}

type Porson struct {
	name string
	age  int
	sex  string
}
```
